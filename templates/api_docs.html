Authentication

All API requests require an API key via the X-API-Key header:

curl -H "X-API-Key: idcard_xxxxxxxx" https://your-domain.com/api/v1/students
Scopes
  • students:read — List and view students
  • students:write — Create, update, delete students
  • templates:read — List and view templates
  • templates:write — Create, delete templates
  • bulk:read — View bulk job status
  • bulk:write — Start and cancel bulk jobs
  • print:read — View print queue and history
  • print:write — Add, cancel, reprint print jobs
  • nfc:read — View NFC encodings
  • nfc:write — Create and execute NFC encodings
  • webhooks:manage — Manage webhook endpoints
  • admin — Full access to all endpoints
