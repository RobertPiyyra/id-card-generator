Tips for Successful Bulk Upload
  • Check template first: Preview the template format to see required dynamic fields.
  • Use correct column names: Column names must match exactly (case-insensitive, but consistency is key).
  • Prepare photos in advance: Collect all photos in one folder and name them consistently.
  • Use relative paths: Photo paths should be relative to your project root (e.g., photos/student_id.jpg).
  • Check for duplicates: Use the "Check for Duplicates" button before processing to avoid double entries.
  • Start with a small test: Test with 2-3 rows first to ensure everything works.
  • CSV formatting: Use UTF-8 encoding to preserve special characters (Urdu/Hindi/Arabic).
Pro tip: After uploading, go to Data Management to verify imported records before generating bulk cards.
Supported Formats & Troubleshooting
Accepted file types
  • .csv (UTF-8 recommended)
  • .xlsx (Excel 2007+)
  • .xls (Legacy Excel)
Photo Requirements
  • Format: JPG, PNG, JPEG
  • Max size: 2MB per image (recommended)
  • Resolution: At least 300x300px for sharp ID cards
  • File names should avoid spaces or special characters
