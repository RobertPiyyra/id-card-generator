About ID Card Generator

A production-ready, multi-school smart ID card generation platform.

Key Features
  • Multi-language support (English, Urdu, Hindi, Arabic) with full RTL
  • CorelDRAW-compatible PDF export (editable + print modes)
  • Drag-and-drop visual card designer
  • Bulk ID card generation from Excel
  • QR codes, barcodes, and NFC chip encoding
  • REST API with webhook support
  • Print queue management
  • Face detection for photo cropping
  • Template version control with snapshots
Tech Stack

Python 3.13 • Flask • SQLAlchemy • PostgreSQL/SQLite • Fabric.js • Pillow • ReportLab
